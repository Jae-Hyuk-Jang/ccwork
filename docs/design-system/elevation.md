# Elevation & Depth

전체 개요는 [`README.md`](README.md) 참고.

## 톤 레이어링 원칙

일반 카드에는 그림자를 쓰지 않는다. `surface_container`(#eaeff1) 섹션 위에 `surface_container_lowest`(#ffffff) 카드를 얹어 "부드러운 들림"을 만든다. 종이가 자연스럽게 겹친 것처럼 보이는 톤 레이어링 방식이다.

## 앰비언트 그림자

떠 있는 효과가 꼭 필요한 경우(예: "새 글" 팝오버):

- **Blur**: 24px ~ 40px
- **Opacity**: `on_surface`(#2b3437)의 6%
- **색상**: 액센트 컬러를 살짝 섞어 팔레트 일관성 유지

## "고스트 보더" 폴백

접근성 등의 이유로 비슷한 배경 위에서도 컨테이너 경계가 꼭 필요한 경우에만 **고스트 보더**를 쓴다.

- **토큰**: `outline_variant`(#abb3b7) **15% 불투명도**. 경계가 아니라 "경계의 암시" 정도여야 한다.
