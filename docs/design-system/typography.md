# 타이포그래피: 에디토리얼 권위

전체 개요는 [`README.md`](README.md) 참고.

**Inter**만 쓰되, 스케일과 굵기를 조절해 권위 있는 위계를 만든다.

| 레벨               | 토큰          | 크기    | 세부 규칙                                                          | 용도                                     |
| ------------------ | ------------- | ------- | ------------------------------------------------------------------ | ---------------------------------------- |
| Display (선언)     | `display-lg`  | 3.5rem  | letter-spacing `-0.02em` — 밀도 있고 맞춤 제작된 느낌              | 랜딩 순간 전용                           |
| Headline (통찰)    | `headline-md` | 1.75rem | line-height `1.4` — 여유로운 가독성                                | TIL 항목 제목                            |
| Body (지식)        | `body-lg`     | 1rem    | 본문은 `on_surface_variant`(#586064), 강조는 `on_surface`(#2b3437) | 장문 읽기, 눈의 피로도 감소              |
| Label (메타데이터) | `label-md`    | 0.75rem | 항상 `uppercase` + letter-spacing `+0.05em`                        | 기능적 메타데이터를 서사적 콘텐츠와 구분 |
