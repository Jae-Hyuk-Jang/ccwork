# 스페이싱 로직

전체 개요는 [`README.md`](README.md) 참고.

그리드는 엄격한 **1.4rem (`spacing.4`)** 리듬을 따른다.

| 용도                                  | 토큰         | 값                                                                                          |
| ------------------------------------- | ------------ | ------------------------------------------------------------------------------------------- |
| 세로 리듬 (헤드라인 → 본문)           | `spacing.2`  | 0.7rem                                                                                      |
| 리스트 아이템 간격 / 기본 컨텐츠 간격 | `spacing.4`  | 1.4rem                                                                                      |
| 섹션 간격 (대형 레이아웃 블록)        | `spacing.10` | 3.5rem — "Atmospheric"한 느낌 유지                                                          |
| 입력 라벨 ↔ 입력창 간격               | `spacing.1`  | 원문에 정확한 값이 명시되지 않음 — 실제 구현 시 `spacing.4`(1.4rem)의 축소 단위로 확정 필요 |
